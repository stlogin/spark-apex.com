# Spark Apex

Corporate website for IT Solutions. Built with modern components based on shadcn/ui.

## Tech Stack

- **Next.js 16** - React Server Components
- **React 19** - Latest React
- **Tailwind CSS** - Utility-first CSS
- **shadcn/ui** - Accessible UI components
- **Radix UI** - Headless UI primitives
- **TypeScript** - Type safety

## Directory Structure

```
src/
├── app/                    # Next.js App Router
├── components/
│   ├── prismui/           # Custom components
│   ├── sections/          # Page sections
│   └── ui/                # shadcn/ui components
└── lib/                   # Utilities
```

## Development

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Type check
pnpm tsc --noEmit

# Lint
pnpm lint
```
